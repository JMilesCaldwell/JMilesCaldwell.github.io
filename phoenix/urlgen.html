Localized URL Handle Generator

Inputs:
A) Shopify skeleton (.xlsx) - for product identification
B) SKU → Handle map (.xlsx) - navigation file
C) SKU + Translated Titles (.xlsx) - Column A=SKU, Column B=Title
D) Output template (.csv) - CSV with 'Type,Identification,Field,Locale,Market,Status,Default content,Translated content'

Output: File D with localized URL handles filled in "Translated content" column.
• Handles normalized (accents removed, spaces→dashes)
• Specs removed unless needed for uniqueness
• Duplicates resolved with minimal spec info
